🚀 How to Run Locally
1. Clone the repository
git clone https://github.com/Geethika_13/Fast-Rag.git cd [Fast-Rag]

2. Install Dependencies
pip install -r requirements.txt

3.Setup API Key
Create a file named secret_api_keys.py in the root folder and add your Hugging Face Token: huggingface_api_key = "your_hf_token_here"

4. Run the app
